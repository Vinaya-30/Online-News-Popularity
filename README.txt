The data is downloaded for the UCI website and the link is as follows
https://archive.ics.uci.edu/ml/datasets/Online+News+Popularity

The zip folder has the data, the html and the ipynb file. The ipynb file can be
executed using the jupyter notebook

To run the code run all the cells. As the data file is in the folder itself no
need to mention the path of the file.

The code uses sklearn, seaborn, numpy and matplotlib libraries.
To install the libraries use pip install "library" name.
